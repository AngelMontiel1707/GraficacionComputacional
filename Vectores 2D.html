vecs: Una lista de vectores que se desean graficar.

cols: Una lista de colores, donde cada color corresponde a uno de los vectores en la lista vecs.

Se crea una nueva figura para el grfico con un tamao especfico de 5x5 pulgadas. Esto proporciona un rea donde los vectores sern visualizados.

La funcin itera sobre la lista de vectores vecs utilizando un bucle

Para cada ndice en la lista de vectores, se obtiene el vector correspondiente.

Se dibuja un vector en el grfico comenzando desde el origen (0,0) y extendindose hasta el punto definido por el vector actual.

El color del vector se asigna segn la lista cols, de modo que cada vector tiene el color correspondiente de la lista.

def grafvecs(vecs, cols):
    plt.figure(figsize=(5,5))
    # vecs: lista de vectores
    for i in range(len(vecs)):
        vec = vecs[i]
        plt.quiver(0,0, vec[0], vec[1], color = cols[i], 
                   angles='xy', scale_units='xy', scale=1)

grafvecs: Define una funcin para graficar mltiples vectores con colores especficos.

plt.figure: Crea una nueva figura de grfico con tamao especfico.

Bucle for: Itera sobre los vectores y los grafica con los colores especificados.
