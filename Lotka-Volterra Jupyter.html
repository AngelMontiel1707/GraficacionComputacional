Modelo Lotka-Volterra¶

¿Que es?¶

El modelo Lotka-Volterra, tambin conocido como el modelo depredador-presa, es un conjunto de ecuaciones diferenciales que describe la dinmica entre dos especies biolgicas que interactan, tpicamente un depredador y una presa. Fue desarrollado independientemente por Alfred Lotka en 1925 y Vito Volterra en 1926.

Se importan 3 librerias las cuales son:

Matplotlib (matplotlib.pyplot as plt) para crear grficos.

Random (random) para generar nmeros aleatorios.

Numpy (numpy) para realizar clculos matemticos y manipular matrices.

Con estos parmetros, se construye y ejecuta una simulacin del modelo Lotka-Volterra

Se definen las poblaciones iniciales de la presa (x) y del depredador (y). Se establecen los parmetros como las tasas de crecimiento, interaccin, y decrecimiento para modelar la dinmica de las poblaciones. Simulacin:

Se utiliza un bucle de tiempo que avanza en pasos pequeos (definidos por dt) desde 0 hasta max_time. En cada paso de tiempo, se calculan las nuevas poblaciones de presa y depredador utilizando las ecuaciones diferenciales que dependen de los parmetros a, b, c, y e.

a = 0.7: Tasa de crecimiento de la poblacin presa (x).

b = 0.5: Tasa de interaccin negativa entre la presa (x) y el depredador (y).

c = 0.3: Tasa de decrecimiento de la poblacin depredadora (y).

e = 0.2: Tasa de interaccin positiva entre el depredador (y) y la presa (x).

dt = 0.001: Paso de tiempo para la simulacin.

max_time = 100: Tiempo mximo de la simulacin.

Estos parmetros se utilizan para definir la dinmica entre las poblaciones de presas y depredadores en el modelo Lotka-Volterra.

c = 0.3  # Tasa de decrecimiento de la poblacin y
e = 0.2  # Tasa de interaccin positiva entre las poblaciones x e y
dt = 0.001  # Paso de tiempo
max_time = 100  # Tiempo mximo de simulacin

# Inicializacin del tiempo y las poblaciones
t = 0   # Tiempo inicial
x = 3.0  # Poblacin inicial de x
y = 0.5  # Poblacin inicial de y

y_list = []

while t < max_time:
    # Calcular nuevos valores de t, x, y

    y = y + (-c * y + e * x * y) * dt

    # Almacenar los nuevos valores en las listas
    t_list.append(t)
    x_list.append(x)
    y_list.append(y)
